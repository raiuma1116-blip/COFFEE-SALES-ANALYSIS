# COFFEE-SALES-ANALYSIS
## Project Overview

The project entails carrying out analysis of the sales of coffee using Microsoft Excel, with the aim of creating key insights based on interactive dashboards. This is with the aim of changing the way raw sales data relates to business decision-making. The dashboard will illustrate sales trends, product sales, and customer behavior using the built-in analytical and visualization capabilities of Microsoft Excel.
